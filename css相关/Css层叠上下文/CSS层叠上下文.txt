层叠上下文的创建
	1.html根元素天生就具备层叠上下文
	2.声明了position:relative | absolute | fixed, 并设置了z-index
	3.css3属性声明会创建层叠上下文条件：
		1.z-index不为auto的值
		2.opacity 透明会创建层叠上下文
		3.transform 会创建层叠上下文
		4.filter 滤镜元素会创建层叠上下文
		
层叠准则
	1.谁大谁上   在同一个层叠上下文中，具有同样的层叠水平标识的话 大的覆盖小的
	2.后者居上   当层叠水平一直,层叠顺序相同,后面的覆盖前面的
	

设置了transform属性的元素的子元素如果有声明固定定位的元素的话,会被降级为absolute,就不会一直固定在顶部
