组件： vue中最重要的特性[]
	1.全局组件
		Vue.component("组件名", {预定义的对象列表});
		全局组件的注册 [组件的注册必须在实例实例化之前]
		
	2.局部组件 [只能在父级的作用域中使用] 局部的组件通过父级组件的实例添加
		var vm = new Vue({
			components: {
				"first-child": {
				},
				"seced-child": {
				}
			}
		})


在定义组件的时候data属性,应该是一个对象这是唯一一个要注意的地方,[为了防止多个应用同一个数据源]
